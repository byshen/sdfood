# San Diego food (圣地亚哥美食)

## 标准

Price (before pandemic) （include 15% tips)

Ratings (个人的口味标准)

- 3 - Awesome 很好吃
- 2 - Good choice 好吃
- 1 - Average cooking 和我自己做的差不多
- 0 - Never come again 神坑，千万不要去
